## To run in local machine 

### > Run this commands 

1. `git clone git@github.com:devbipu/Laravel-Blog.git`

2. `cd Laravel-Blog`

3. `composer install`

5. `cp .env.example .env` & `php artisan key:generate`

6. `php artisan migrate` or `php artisan migrate:refresh --seed`
